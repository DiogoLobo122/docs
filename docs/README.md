### Start contributing to `docmoa`

<code-group>
<code-block title="NPM">
```bash {2,5-6,9}
# git clone
git clone https://github.com/docmoa/page.git

# npm install
cd page
npm install

# start VuePress writing
npm run dev
```
</code-block>

<code-block title="YARN">
```bash {2,5-6,9}
# git clone
git clone https://github.com/docmoa/page.git

# npm install
cd page
yarn install

# start VuePress writing
yarn vuepress dev
```
</code-block>
