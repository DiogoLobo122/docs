# Contribute

docmoa에 문서 기여하기위한 가이드를 설명합니다. 일반적인 github 상에서의 코드 기여 방식과 거의 비슷합니다.

## github fork
문서는 모두 git으로 관리됩니다.
```bash
git clone https://github.com/docmoa/page.git
```

또는 github에서 fork하여 별도 관리 후 pull request 하여도 좋습니다. fork 방식은 구분하여 설명합니다.
